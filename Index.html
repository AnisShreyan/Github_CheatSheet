Github Basic Commands Cheat Sheet

Here are some basic github commands to get started with github. This Unofficial Documentation is created for my personol future refernce. It will also be helpful for anyone want to learn the basics of github.

Before starting download git from here => download git

Global Username and User Email

Set/Create or Config Global Username :


                git config --global user.name "Your Username"

Change or Edit Global Username :


                git config --global --edit user.name "Your Username"

Check the Global Username :


                git config --global user.name

Set/Create or Config Global User Email :


                git config --global user.name "Your Username"

Change or Edit Global User Email :


                git config --global --edit user.name "Your Username"

Check the Global User Email :


                git config --global user.name

Git Init

Git init command initialize an empty git repository 


                git init

Git Status

Git status command says the status of the git repository 


                git status

Git Add

Git add command adds the files of the git repository for staging

Staging area is basically a state of the repo what holds the state of git commitments


                git add filename.extention

Git Commit

Git commit command commit the git repository that is stagged


                git commit -m "Your Commit Massage"
