Git Init

Git init command initialize an empty git repository 


                git init

Git Status

Git status command says the status of the git repository 


                git status

Git Add

Git add command adds the files of the git repository for staging

Staging area is basically a state of the repo what holds the state of git commitments


                git add filename.extention

Git Commit

Git commit command commit the git repository that is stagged


                git commit -m "Your Commit Massage"

Git Log

Git Log command reviews and reads a history of everything that happens to a repository


                git log

Git Checkout Commits

Git checkout command switches between differnt commits/stages


                git checkout [commit id]

Git Branch

Git branch command shows the branches of the repository


                git branch

Create a Git Branch

Git branch command [Branch name] create the new branches in the repository


                git branch [Branch Name]

Checkout a Git Branch

Git checkout [branch name] command switches between differnt braches of the repository


                git checkout [Branch Name]

Create & Checkout a new Git Branch

Create a branch and checkout in branch


                git checkout -b [Branch Name]

Mergeing Git Branches

Merge a branch with the current branch that you are checked in


                git merge [Branch Name]

Deleting or Removing Git Branches

Delete or remove a git branch from the repository


                git branch --delete [Branch Name]
